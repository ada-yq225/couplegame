## 快速上手

### 电脑端

**Windows**

1. 克隆或下载本项目
2. 进入 `couplegame` 文件夹
3. 双击 `start-phone.bat`（会调用 `server.ps1` 启动静态服务）
4. 保持终端窗口不要关闭
5. 浏览器打开：**http://127.0.0.1:8765**

**macOS / Linux**

```bash
cd couplegame
python3 -m http.server 8765
```

浏览器访问 **http://127.0.0.1:8765**

### 手机端（同一 WiFi）

1. 电脑先启动本地服务
2. 查看电脑局域网 IP（如 `192.168.1.23`）
3. 手机连同一 WiFi，浏览器打开 `http://192.168.x.x:8765`
4. 建议「添加到主屏幕」，可全屏像 App 一样使用

### 第一次进入

1. 输入双方昵称（TA / 你）
2. 勾选成年自愿确认
3. 进入大厅，选择想玩的模式

### 更新后加载旧版？

Service Worker 会缓存资源。请 **硬刷新**（`Cmd+Shift+R` / `Ctrl+F5`），或清除站点缓存后重开。当前缓存版本见 `sw.js` 中的 `CACHE` 字段。

## 色情 Galgame（Orion）

大厅点击 **色情 Galgame** 进入青藤大学。这是一款 Orion 式校园 Roguelike：24 周四幕学期、30 张地图、每位女主 15 个 H 场景、42+ 特殊玩法，带完整剧情线与攻略系统。

### 开局流程

1. **性癖罗盘** — 选择本轮偏好张力（影响好感与描写风格）
2. **路线计划** — 决定这学期怎么过（单线、多线、夜游、礼物、学业夹缝、纯爱慢热等）
3. **校园生活** — 每天随机事件，遇人、约会、推剧情、开 H

首次进入会自动展开 **指南面板**，顶部黄条为情境提示，可随时点「指南」查看完整攻略。

| lintang | 林晚棠 | 冰山法学少女 | 雨夜、图书馆、法条式暧昧 |
| sutang | 苏糖 | 甜软舞蹈生 | 奶茶、撒娇、兔耳试装 |

### 核心数值

| 数值 | 含义 |
|------|------|
| 体力 | 行动消耗，耗尽需休息 |
| 欲压 | 过高影响检定，谈心/约会可降 |
| 学业 | 周末考核用，过低有惩罚 |
| 魅力 | 撩骚、邀约、偶遇检定 |

| 金钱 | 商店买礼物与道具 |

### 六档关系

陌生 → 留意 → 暧昧 → 点燃 → 肉体 → 沉迷

每升一档解锁更多攻略手段。遇人时可：

- 单独约会、撩她、深夜谈心
- 打开 **攻略手册**（按关系阶段显示可用招式）
- 推进剧情（十二幕主线）
- 带她去做 H（五阶段沉浸：靠近 → 脱衣 → 前戏 → 插入 → 高潮）
- 特殊玩法（脱衣扑克、共浴、镜前、课上遥控等 36+ 种）
- 角色扮演（师生、法庭、女仆、女王、私教、摄影、囚禁、酒吧搭讪等 12 种）

### 纯爱慢热线

开局选 **纯爱羁绊** + **纯爱慢热**，或中途换策略切纯爱：

- 剧情走 `ORION_PURE_STORYLINES`（每位女主 12 幕纯爱剧情）
- H 走温柔缠绵正文，前戏描写偏软
- 专属选项：**牵手散步**、**晚安吻**
- 推剧情可用信任替代性奋（`trust ≥ 2`）；第 4 章起需 `trust ≥ 3`
- H 入口：`好感 ≥ 5` 且 `信任 ≥ 6`（不必有性奋）
- 完美结局：最高好感 ≥ 10、信任 ≥ 8、十二幕通关、关系「沉迷」

### 学期四幕（24 周）

| 幕 | 周次 | 重点 |
|----|------|------|
| 播种 | 1–6 | 留意 → 暧昧，蹲点与偶遇 |
| 点燃 | 7–12 | 告白与第一次 H，推剧情 |
| 沉迷 | 13–18 | 肉体 → 沉迷，信任与谈心 |
| 占有 | 19–24 | 完美结局 / 纯爱结局冲刺 |

第 7 天为周末考核；第 8、16、20 周有喘息周奖励可选。

| 按钮 | 功能 |
|------|------|
| 指南 | 展开/收起攻略指南面板 |
| 保存 | 手动存档（也会自动保存） |

### 存档

## 情欲沉浸 & 深夜快闪

### 情欲沉浸

- 34 篇完整黄文，按地点分类（公共 / 交通 / 室内 / 户外）
- 每篇多段落叙事，读完一段可翻下一段
- 段落末尾附带 **「现在就干」** 色牌指令，可直接照做
- 支持随机一篇、按地点筛选
- 联机模式下主机可选「沉浸」，副机同步阅读

### 深夜快闪

- 随机抽一篇黄文 + 配套极限色牌
- 节奏快、惩罚感强，适合「不想选、就想被安排」的夜晚

## 经典互动玩法

以下为蜜语原始核心玩法，适合两人面对面实操。

### 三种强度

| 强度 | 风格 |
|------|------|
| 🫦 前戏 | 亲吻、爱抚、脱衣、暧昧升温 |
| 🔥 火热 | 口舌、手指、强烈挑逗 |
| 💦 深入 | 进入、体位、同步高潮 |

### 五种基础机制

| 玩法 | 说明 |
|------|------|
| 🃏 卡牌 | 翻牌执行指令，含限时 / 二选一 / 脱一件 / 情欲加码 |
| 🎲 骰子 | 部位 × 动作 × 主动方，随机组合 |
| 🎡 轮盘 | 转盘决定本回合玩法 |
| 🎭 盲选 | 三选一暗牌，选中必须执行 |
| 🎬 情境 | 短剧情设定（雨夜重逢、厕所隔间等） |

### 闯关模式

- 9 章主线 + 1 章隐藏章（Lv.12 且通关一次后解锁）
- 每章若干小关 + Boss 关，按跳过次数评星
- 通关按总星数评定结局 C → SS

### 场景故事

- 34 种地点，每种 4 幕（约 16–20 张情境牌）
- 支持按强度、地点类型筛选
- 完整剧终额外 XP 奖励

### 挑战模式（10 种专题）

极速、闪击、硬核、边缘、感官、同步、告白、骰子王、轮盘狂、马拉松 — 各有独立开场、关卡脚本与结算语。完成奖励 +120 XP。

### 每日挑战

每天 12 关连战，主题随机（骰子之日、边缘之日、不许跳过等），完成 +150 XP，当日不可重复。

### PK 对决

15 回合，先到 300 分或回合结束分高者胜。每回合完成后判定「谁更猛」，输家可能触发口头惩罚。

### 自由模式 & 自定义

自由模式可自由组合玩法与强度；情境模式可指定场景或随机混合。卡牌编辑器可自制指令，支持 JSON 导入导出。

### 文案占位符

| 占位符 | 含义 |
|--------|------|
| `{self}` | 当前主动方 |
| `{other}` | 当前被动方 |
| `{A}` / `{B}` | 开局输入的两个名字 |

## 异地同房（联机）

基于 **PeerJS** 点对点连接，不要求同一 WiFi，但双方需能访问互联网。

**主机**：创建房间 → 记下 6 位房间码 → 等副机加入 → 选模式开始

**副机**：输入房间码加入 → 等主机开始 → 画面自动同步

可联机模式：闯关、自由、PK、每日、场景故事、情欲沉浸。

副机可翻牌、完成、跳过、掷骰等，操作发给主机统一结算。顶部绿条显示连接状态。

## 成长系统

- 完成各模式获得 XP，升级至约 Lv.50
- 升级随机获得道具（情欲加倍、预知、跳过护盾、偷分等）

- 道具背包上限 8 个，每局开始不足 3 个会补充

## 常见问题

**页面打不开？**  
确认本地服务在运行，端口 8765 未被占用；检查防火墙。

**手机访问不了电脑地址？**  
确认同一 WiFi；关闭 VPN；Windows 需允许 PowerShell 入网。

**Galgame 点了没反应？**  
硬刷新清缓存；若界面报错，点「重试刷新」或「返回大厅」再进。

**联机连不上？**  
检查网络与房间码；主机需等副机显示已连接再开始。

**语音没声音？**  
检查大厅语音开关、浏览器标签页静音、系统音量。

**存档丢了？**  
存档在浏览器 localStorage，无痕模式或清除站点数据会丢失。Galgame 与闯关等模式存档相互独立。

**可以两人各玩各的 Galgame 吗？**  
可以，不同设备存档独立；要同步画面请用「异地同房」。

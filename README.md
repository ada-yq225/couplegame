>
> **使用前请务必确认**：双方均为成年人、关系自愿、可随时喊停。本游戏仅为私密情趣辅助工具，请在安全、私密、合法的环境中使用。

## 1. 快速上手

### 1.1 电脑端启动

1. 下载或克隆本项目到本地
2. 进入 `couples-game` 文件夹
3. **双击** `start-phone.bat`
4. 保持弹出的黑色窗口**不要关闭**（关闭即停止服务）
5. 浏览器访问：**http://127.0.0.1:8765**

> 本项目使用 PowerShell 脚本 `server.ps1` 提供静态文件服务，**不需要安装 Python**。

### 1.2 手机端（同一 WiFi）

1. 电脑上先运行 `start-phone.bat`
2. 记下终端里显示的手机地址，形如：`http://192.168.x.x:8765`
3. 手机连**同一 WiFi**，用浏览器打开该地址
4. 建议：浏览器菜单 → **添加到主屏幕**，可像 App 一样全屏打开

### 1.3 第一次进入

1. 输入双方昵称（TA / 你）
2. 勾选「双方成年、自愿，随时可停」
3. 点击 **进入游戏大厅**
4. 在大厅选择想玩的模式即可

## 2. 界面与基本操作

### 2.1 主要界面

| 界面 | 作用 |
|------|------|
| 欢迎页 | 输入名字、确认自愿 |
| 游戏大厅 | 选择模式、设置、成就、编辑器 |
| 章节开场 | 闯关模式每章开始前的介绍 |
| 游戏主界面 | 翻牌 / 掷骰 / 轮盘 / 盲选等 |
| 章节通关 | 显示本章星级，进入下一章 |
| 最终结算 | 本局结束，获得 XP |
| 挑战选择 | 十种挑战专题列表 |
| 场景选择 | 三十四种地点剧情列表，支持强度/地点筛选 |
| 自由模式设置 | 选择玩法与强度（情境模式可选场景） |
| 异地同房 | 创建/加入联机房间 |
| 成就 / 编辑器 | 子页面 |

### 2.2 游戏内按钮

| 按钮 | 功能 |
|------|------|
| **做完了** | 确认本关任务已完成，进入下一关 |
| **跳过** | 跳过本关（部分模式禁止跳过） |
| **下一张** | 与跳过类似，快速换下一张 |
| **← 返回** | 退出本局（闯关会保存进度） |
| **🎮** | 仅自由模式可用：切换卡牌/骰子/轮盘/盲选 |

### 2.3 卡牌操作

- **点击卡牌** 或 **向上滑动** → 翻开卡牌
- 二选一牌：翻开后再点选项
- 限时节：翻开后自动开始倒计时，语音可播报

### 2.4 文案占位符

卡牌与指令中的变量会自动替换为双方名字：

| 占位符 | 含义 |
|--------|------|
| `{self}` | 当前主动方 |
| `{other}` | 当前被动方 |
| `{A}` / `{B}` | 固定为开局输入的两个名字 |
| `{time}` | 当前限时秒数 |

## 3. 通用规则与数值系统

### 3.1 三种强度

| 强度 | 含义 | 出现场景 |
|------|------|----------|
| 🫦 **前戏** | 亲吻、爱抚、脱衣、暧昧升温 | 第 1 章、感官挑战、部分每日 |
| 🔥 **火热** | 口舌、手指、强烈挑逗 | 多数章节、PK、极速挑战 |
| 💦 **深入** | 进入、体位、同步高潮 | 后期章节、硬核/闪击挑战 |

### 3.2 三种卡牌类型

| 类型 | 含义 |
|------|------|
| **服侍 TA** | 当前主动方取悦对方 |
| **享受 TA** | 当前主动方躺好接受 |
| **一起** | 双方同步完成，不分主被动 |

### 3.3 界面数值条

**热度（0–100%）**
- 每完成一关 +10 起，连击越高加得越多
- 满 100 时触发「情欲满溢」奖励指令，随后清零

**情欲值（双方各 0–100%）**
- 闯关 / PK / 每日 / 挑战模式显示
- 完成关卡时主动方加得多，被动方也会涨
- 双方同时满 100 可解锁成就

**比分（A : B）**
- 自由模式与 PK 等会累计
- 完成关卡：主动方 +10 起，连击有加成

**连击**
- 连续完成不跳过则累加
- 连击越高，热度与得分加成越大
- 跳过则连击清零

### 3.4 三种难度（大厅右上角切换）

| 难度 | 效果 |
|------|------|
| 温柔 | 情欲/得分 ×0.8–0.9，可跳过，随机事件少 |
| 标准 | 基准数值 |
| 狂野 | 情欲/得分加成，**禁止跳过**，随机事件多 |

### 3.5 随机事件（闯关/自由等）

进行中有概率触发顶部横幅事件，例如：

- 情欲爆发、强制换位、得分翻倍、热浪
- 本关变盲选、限时 60 秒、必须同步完成
- 双倍赌局（完成加分，跳过扣分）
- 情境降临、禁语、脱衣轮盘、慢动作等

> **挑战模式不会触发随机事件**，以保证关卡脚本稳定。

### 3.6 特殊卡牌（混在卡牌堆里）

| 类型 | 说明 |
|------|------|
| ⚡ **限时** | 翻开后倒计时，时间到须停 |
| 🔀 **二选一** | 两项择一执行 |
| 👗 **脱一件** | 根据服侍/享受/一起有不同文案 |
| 💥 **情欲加码** | 下一张强度升一级，且不可跳过 |

## 4. 五种基础玩法详解

以下五种玩法会出现在**闯关关卡**、**自由模式**、**挑战关卡**、**每日任务**中。

### 4.1 🃏 卡牌模式

**流程**
1. 显示卡背 → 点击或上滑翻开
2. 阅读指令并执行
3. 点「做完了」进入下一关

**牌库内容**
- 三类基础牌（服侍 / 享受 / 一起）× 三档强度，每类约 20 条
- 特殊牌：限时、二选一、脱一件、情欲加码
- 情境牌：角色扮演、挑战、悄悄话（混入牌堆）
- 若开启「混合自定义卡牌」，你编辑的牌也会进入

**技巧**
- 同局内尽量不重复最近抽过的文案
- 限时节配合语音倒计时更有氛围

---

### 4.2 🎲 骰子模式

**流程**
1. 点击「掷骰子」
2. 系统随机 **部位 × 动作 × 主动方**
3. 按结果显示的句子执行
4. 点「做完了」

**部位示例（随强度变化）**
- 前戏：嘴唇、耳垂、锁骨、大腿内侧……
- 火热：乳头、腿根、会阴、臀缝……
- 深入：G 点/前列腺、结合处、腰窝……

**动作示例**
- 前戏：亲吻、舔舐、指尖打圈、慢慢脱衣……
- 火热：吸吮、口舌侍奉、揉捏、边做边描述……
- 深入：深顶、骑乘律动、后入、同步高潮……

**显示格式**
> `{主动方} 用「{动作}」对待 {被动方} 的「{部位}」`

---

### 4.3 🎡 轮盘模式

**流程**
1. 点击「转轮盘」，等待转盘停下
2. 按结果执行对应指令
3. 点「做完了」

**可能结果**

| 结果 | 效果 |
|------|------|
| 服侍 / 享受 / 一起 | 随机抽一条对应类型卡牌执行 |
| 掷骰 | 等同于掷一次骰子 |
| 限时 60 秒 | 随机限时指令 + 倒计时 |
| 脱一件 | 脱衣惩罚指令 |
| 情欲加码 | 热度 +30，奖励类指令 |
| 情境 / 挑战 / 悄悄话 | 抽取情境类文案 |
| 换位 | 主动方立刻交换 |

---

### 4.4 🎭 盲选模式

**流程**
1. 系统暗置 3 张牌（内容不提前公开）
2. 你们选其中一张翻开
3. **选中就必须做**，不能反悔
4. 点「做完了」

适合喜欢「冒险感」、愿意接受未知指令的情侣。

---

### 4.5 🎬 情境模式

情境牌分四类，牌面颜色与标签不同：

| 类型 | 内容风格 |
|------|----------|
| 🎬 **情境** | 短剧情设定，如「雨夜重逢」「厕所隔间」「万米高空」 |
| 🎯 **挑战** | 带惩罚/限时的 dare 任务 |
| 👂 **悄悄话** | 耳边描述欲望、感受 |
| 💬 **告白** | 边亲热边说实话、坦白幻想 |

**自由模式**中选「情境」即全程抽情境牌，还可指定 **34 种场景** 之一；**场景故事**模式则按选定地点完整演绎四幕剧情。**挑战/每日**中也会穿插情境关卡。

## 5. 大厅模式详解

大厅共有 **8 个入口**，下面逐一说明。

---

### 5.1 🎮 闯关模式

**适合**：想有完整「剧情推进感」、从零到高潮循序渐进的情侣。

**结构**
- 共 **9 章主线** + **1 章隐藏章**
- 每章若干小关 + **1 个 Boss 关**
- Boss 必须完成，不可跳过
- 章末按跳过次数评星：0 跳过 = 3 星，1 跳过 = 2 星，更多 = 1 星

**流程**
1. 大厅 → 闯关模式
2. 阅读章节介绍 → 进入章节
3. 逐关完成 → Boss 战 → 章节结算
4. 选「下一章」或「回大厅」
5. 可随时点「继续闯关进度」从存档恢复

**存档**
- 自动保存在浏览器本地
- 记录章节、回合、情欲、得分、星级等
- 通关全部章节后存档清除

**通关奖励**
- 每章 XP + 星级加成
- 全部通关根据总星数评定结局（C → SS）
- 通关后补充道具

详见 [第 15 章 章节全览](#15-闯关章节全览与隐藏章)。

---

### 5.2 🎯 挑战模式

**适合**：时间有限、想玩固定主题、或想体验高强度脚本的情侣。

- 共 **10 种专题**，每种有独立开场白、关卡脚本、结算语
- 每关顶部显示 **📋 本关提示（brief）**
- 卡牌关常带 **固定任务文案**，不是纯随机
- 完成奖励 **+120 XP**

详见 [第 7 章](#7-挑战模式--十种专题)。

---

### 5.3 📅 每日挑战

**适合**：每天玩一点、保持新鲜感。

- 每天 **0 点刷新**（按本地日期）
- **12 关**连战，混合多种玩法
- 每日有随机 **主题 modifier**（如骰子之日、边缘之日、不许跳过等）
- 每关顶部有一条 **今日提示**
- 完成奖励 **+150 XP**，当日不可重复领取

详见 [第 8 章](#8-每日挑战)。

---

### 5.4 ⚔️ PK 对决

**适合**：喜欢竞争、互不服输的情侣。

- **15 回合**，先到 **300 分** 或回合结束时分高者胜
- 每回合随机玩法（卡牌/骰子/盲选/轮盘）
- 回合结束后弹出 **「谁更猛？」** 由你们诚实判定
- 胜者 +35 / 败者 +8；平局各 +15
- 输家可能触发口头惩罚（随机一句）
- 道具「偷分」仅 PK 可用

详见 [第 9 章](#9-pk-对决)。

---

### 5.5 📖 场景故事

**适合**：想要完整「地点 + 剧情」沉浸感、从暧昧循序渐进到深入的情侣。

- 共 **34 种地点剧情**，每种 **4 幕**（约 16～20 张情境牌）
- 每幕有开场旁白 + 多轮情境/挑战/悄悄话/告白牌
- 支持按 **强度**（偏前戏 / 偏火热 / 偏深入）与 **地点类型** 筛选
- 每幕结算按跳过次数评星，获得 **55～85 XP**；完整剧终额外 **+180 XP**
- 记录「上次玩过」的场景，方便下次继续选

**流程**
1. 大厅 → 场景故事
2. 浏览场景列表，点选想演的地点
3. 阅读每幕开场 → 逐张完成情境牌
4. 幕间结算 → 下一幕 → 故事结局

详见 [第 6 章 场景一览](#6-场景故事--三十四种地点)。

---

### 5.6 🃏 自由模式

**适合**：不想被章节绑定、随心所欲组合玩法。

1. 大厅 → 自由模式
2. 选择 **玩法**（卡牌/骰子/轮盘/盲选/情境）
3. 选择 **强度**（前戏/火热/深入）
4. 若选 **情境**，可额外指定 **场景**（或随机混合全库）
5. 开始游戏，无固定关数，牌堆耗尽即结束

- 游戏中点 **🎮** 可切换玩法（重新洗牌）
- 得分、热度、连击照常计算
- 结束奖励 +40 XP

详见 [第 10 章](#10-自由模式与自定义)。

---

### 5.7 ✨ 自定义关

**适合**：想只玩自己写的指令。

- 需先在 **卡牌编辑器** 中添加至少一张卡
- 只从你的自定义牌库抽牌
- 强度固定为「火热」

---

### 5.8 📡 异地同房

**适合**：两台设备不在同一地点，但想同步看同一张牌。

- 基于 **PeerJS** 点对点连接，不经过自建服务器
- **主机**创建房间，**副机**输入 6 位房间码加入
- 主机选择模式后开始，副机画面实时同步
- 副机可翻牌、完成、跳过、掷骰等（指令发给主机执行）
- 联机自由模式同样支持 **情境 + 指定场景**

详见 [第 11 章](#11-异地同房联机)。

### 6.3 使用建议

- **第一次玩**：推荐 🌧 雨夜重逢、🏨 酒店一夜、🔥 冬夜壁炉——节奏完整、地点熟悉
- **想要刺激**：试 🚻 厕所隔间、🛗 电梯、🎬 电影院等公共场所类（记得确保真实环境也安全私密）

进入：**大厅 → 挑战模式 → 选择专题**

| 💀 硬核 | 15 | 无 | **禁止** | 深入 | 抽到什么做什么 |

### 7.1 ⏱ 极速（12 关）

**开场**：「倒计时一响就必须动，磨蹭的人会落在后面。」

**关卡节奏示例**
1. 热身 · 卡牌 · 45s 脱衣亲吻
2. 快掷 · 骰子
3. 口舌 · 卡牌 · 45s 脖颈到胸口
4. 命运 · 轮盘
5. 盲选 · 三选一
6. 同步 · 卡牌 · 45s 贴身磨蹭
7. …直至终冲

**适合**：前置暧昧已经够了，想快速进入状态。

---

### 7.2 ⚡ 闪击（10 关）

**开场**：「每关只有三十秒，做不到也要做。」

- 每关 **30 秒**硬限时
- **禁止跳过**
- 强度「深入」，任务直白

**适合**：两人都已经很熟了，想要极限压迫感。

---

### 7.3 💀 硬核（15 关）

**开场**：「不许跳过，不许讨价还价。」

- 关卡含：服从跪吻、深吻、口舌侍奉、进入、骑乘、后入、同步高潮等
- 穿插盲选、骰子、情境挑战
- Boss 级任务：必须做到两人一起高潮

**适合**：信任度高、愿意完全交付的一对。

---

### 7.4 🌊 边缘（12 关）

**开场**：「忍住别高潮，谁先失控谁下一轮全程服侍。」

- 核心机制：**快到顶 → 停下 → 重复 → 最后一起释放**
- 多数关 60 秒计时
- 禁止跳过

**适合**：想拉长快感、练习自控力。

---

### 7.5 👁 感官（14 关）

**开场**：「慢一点，细一点，感受比速度重要。」

- 蒙眼触摸、慢吻、隔衣按压、边缘、舔舐、关灯探索
- 强度以前戏为主
- 强调呼吸、触觉、耳语

**适合**：想要浪漫、细腻、不急着「进入主题」的夜晚。

---

### 7.6 🔗 同步（14 关）

**开场**：「今晚没有『我一个人来』，每件事都要一起完成。」

- 关卡绝大多数为「一起」类任务
- 牵手、同步磨蹭、互摸、对视手淫、同步呼吸、同步高潮

**适合**：强调「双人感」、默契和陪伴。

---

### 7.7 💬 告白（10 关）

**开场**：「边亲热边说实话，害羞可以，撒谎不行。」

- 坦白秘密、边吻边说、说出渴望、回忆第一次
- 情境 + 卡牌 + 告白牌混合

**适合**：想加深情感连接，不只是身体互动。

---

### 7.8 🎲 骰子王（15 关）

**开场**：「点数决定部位，动作决定玩法，别指望会重复。」

- 15 关几乎全是掷骰
- 部位 + 动作组合随机

**适合**：喜欢随机感、每次组合都不同。

---

### 7.9 🎡 轮盘狂（16 关）

**开场**：「命运替你们做决定，转到什么就玩什么。」

- 16 关以轮盘为主，少量骰子/盲选/情境调剂

**适合**：把决定权交给「运气」，增加未知感。

---

### 7.10 🏃 马拉松（20 关）

**开场**：「二十关连战，中途想停可以跳过，但别轻易认输。」

- 最长挑战，从前戏逐步过渡到深入
- 混合全部五种基础玩法
- 第 20 关为终章同步高潮

**适合**：周末有充足时间、想打一整场「副本」。

## 8. 每日挑战

- 每天 **1 次**完成机会（本地日期刷新）

## 9. PK 对决

### 9.1 胜利条件

- **15 回合**内先达到 **300 分** → 立即获胜
- 或 15 回合后 **分高者** 获胜
- 同分 → 平局

### 9.2 单回合流程

1. 随机一种玩法进行一回合
2. 完成后点「做完了」
3. 弹出 **谁更猛？** → 选 A / 选 B / 平局
4. 按判定加分，进入下一回合

### 9.3 计分

| 结果 | 分数 |
|------|------|
| 被判「更猛」 | +35 |
| 被判「输了」 | +8 |
| 平局 | 双方各 +15 |

### 9.4 输家惩罚（随机一句）

例如：「输的人用嘴服侍赢的人两分钟」「输的人脱一件，让赢的人摸一分钟」等。

### 9.5 PK 专属道具

- **偷分 🗡**：从领先方偷 20 分（仅 PK 可用）

## 11. 异地同房（联机）

### 11.1 前提

- 双方设备均需 **能访问互联网**（PeerJS 信令需要）
- **不要求**同一 WiFi
- 建议双方先各自打开游戏页面

### 11.2 连接步骤

**主机（创建房间）**
1. 大厅 → 异地同房 → **创建房间**
2. 记下 6 位 **房间码**
3. 等副机加入（显示「已连接」）
4. 选择模式：闯关 / 自由 / PK / 每日
5. 开始游戏

**副机（加入房间）**
1. 大厅 → 异地同房
2. 输入房间码 → **加入房间**
3. 回到大厅等待主机开始
4. 主机开始后画面自动同步

### 11.3 同步内容

- 当前界面（章节/游戏/结算等）
- 卡牌文字、骰子结果、轮盘结果
- 倒计时、比分、情欲、热度
- PK 判定弹窗

### 11.4 副机可执行的操作

翻牌、做完了、跳过、掷骰、转轮、盲选、PK 判定、章节推进（发给主机）

> **主机控制主流程**：副机操作会发送给主机，由主机统一结算。

### 11.5 状态栏

游戏界面顶部绿色条显示：`同房 XXXXXX · 已连接`

## 12. 成长系统：等级、道具、成就

### 12.1 等级与 XP

- 完成关卡、通关、挑战、每日、PK 等均可获得 XP
- XP 满后升级，最高约 Lv.50
- 升级可能随机获得道具（背包上限 8 个）

### 12.2 道具一览（游戏中点击图标使用）

| 道具 | 效果 |
|------|------|
| 💗 情欲加倍 | 本回合情欲获取 ×2 |
| 👁 预知 | 预览下一张指令内容 |
| 👑 强制服侍 | 对方本回合必须服侍你 |

## 16. 常见问题

**Q：页面打不开？**  
A：确认 `start-phone.bat` 窗口未关闭；检查防火墙是否拦截 8765 端口。

**Q：手机访问不了电脑地址？**  
A：确认同一 WiFi；Windows 防火墙允许 PowerShell；尝试关闭 VPN。

**Q：联机连不上？**  
A：检查网络；确认房间码正确；主机需等副机显示已连接再开始。

**Q：语音没声音？**  
A：检查大厅语音开关、浏览器标签页是否静音、系统音量。

**Q：更新后还是旧版？**  
A：Ctrl+F5 强制刷新，或清除浏览器缓存后重开。

**Q：存档丢了？**  
A：存档在浏览器 localStorage，清除站点数据会丢失；勿用无痕模式长期玩。

**Q：可以两个人各用手机本地玩吗？**  
A：可以，不同设备存档独立；要同步画面请用「异地同房」。

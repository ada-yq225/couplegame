# 蜜语

成年情侣向的本地私密互动游戏。纯前端，无需安装，浏览器打开即可玩；支持手机同屏、异地联机、语音倒计时与 PWA 离线缓存。

> 双方须成年、自愿参与，可随时暂停或退出。

## 功能概览

### 游戏模式

| 模式 | 说明 |
|------|------|
| **闯关** | 9 章主线 + 隐藏章，Boss 战、星级评分、多结局 |
| **挑战** | 10 种专题（极速、闪击、硬核、边缘、感官、同步、告白、骰子王、轮盘狂、马拉松），每关独立文案 |
| **每日挑战** | 按日期生成当日任务与 modifier |
| **PK 对决** | 15 回合争分，人工判定胜负 |
| **自由模式** | 卡牌 / 骰子 / 轮盘 / 盲选 / 情境 |
| **自定义关** | 使用编辑器自制卡牌 |
| **异地同房** | PeerJS P2P 联机，主机控流程，副机同步画面 |

### 系统

- 等级与 XP、24+ 成就、8 种道具
- 情欲双条、热度、连击、随机事件
- 语音倒计时（Web Speech API，可开关）
- 卡牌编辑器（增删、导入/导出 JSON）
- 闯关自动存档
- 三档难度（温柔 / 标准 / 狂野）

## 快速开始

### 电脑本地

1. 进入项目目录
2. 双击 `start-phone.bat`（内部调用 `server.ps1`）
3. 浏览器打开 `http://127.0.0.1:8765`

### 手机同 WiFi

1. 电脑上运行 `start-phone.bat`
2. 手机浏览器访问终端里显示的 `http://你的电脑IP:8765`
3. 可选：浏览器「添加到主屏幕」当 App 使用

### 无 Python 环境

项目使用 PowerShell `TcpListener` 提供静态文件服务，Windows 开箱可用。

## 联机说明

1. 大厅 → **异地同房**
2. 一方 **创建房间**，把 6 位房间码发给对方
3. 另一方 **加入房间**
4. 主机选择模式（闯关 / 自由 / PK / 每日）后开始

联机依赖 PeerJS CDN，需要联网。主机控制流程，副机可翻牌、完成、跳过等操作并同步画面。

## 隐藏章解锁

- 达到 **Lv.12**
- 闯关模式 **通关至少 1 次**

通关第九章后可进入「隐秘之夜 · 神话」。

## 项目结构

```
couples-game/
├── index.html        # 主页面
├── app.js            # 核心逻辑
├── data.js           # 基础卡牌与配置
├── content-extra.js  # 扩充内容包
├── challenges.js     # 挑战模式关卡脚本
├── profile.js        # 存档、成就、每日挑战
├── sync.js           # 联机同步
├── voice.js          # 语音倒计时
├── styles.css
├── sw.js             # Service Worker 离线缓存
├── server.ps1        # 本地 HTTP 服务
└── start-phone.bat   # 一键启动
```

## 技术栈

- 原生 HTML / CSS / JavaScript
- localStorage 存档
- PeerJS 1.5.2（联机）
- Web Speech API（语音）
- PWA（manifest + service worker）

## 隐私

数据保存在本机浏览器，不上传服务器。联机为点对点连接，不经过自建后端。
